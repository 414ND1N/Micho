# TOFFU
## Onanībando

[![Build Status](https://travis-ci.org/joemccann/dillinger.svg?branch=master)](https://travis-ci.org/joemccann/dillinger)


Bot de discord con un prefix de *tf.* para los comandos, compatible con los comandos slash

### Comandos Generales
* ping
* pagina
* codigo

### Comandos Música
* djpanas
* play
* pausar
* skip
* previous
* stop
* resumir
* volumen <*numero*>
* mezclar
